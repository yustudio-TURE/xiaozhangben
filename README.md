# 小账本 - 个人语音记账应用

一个 PWA 网页应用，专为 iPhone 设计。通过语音输入消费内容，自动分类，提供月度/年度统计。

## 如何使用

### 方式一：iPhone Safari 打开（推荐）
1. 将整个文件夹上传到 GitHub Pages 或其他静态服务器
2. iPhone 用 Safari 打开网址
3. 点击底部「分享」→「添加到主屏幕」
4. 桌面出现「小账本」图标，点击即可使用（像原生 App 一样）

### 方式二：本地测试
1. 双击 `index.html` 在浏览器中打开
2. 语音功能需要 HTTPS 环境，本地可用 Chrome DevTools 模拟
3. 或安装 Node.js 后运行: `npx serve .`

## 功能

- 🎤 **语音输入**: 说出消费内容，自动识别
- 🤖 **智能分类**: 自动归入餐饮/交通/购物/娱乐等 8 个类别
- 📊 **统计图表**: 月度/年度支出占比和趋势图
- 📱 **PWA**: 可添加到 iPhone 主屏幕，离线使用
- 💾 **本地存储**: 数据保存在浏览器中，不经过任何服务器
